# tex-template
A easy to use LaTeX tool for compiling homeworks and such.

Available commands:
  - `all`
      compile all problem sets

  - `ps <ps number>`
      compile a specific problem set

  - `new`
      create a new problem set

  - `clean`
      clean the build dir

Example syntax:
    `./tex.sh ps 2`
    This will compile `./src/PS_2.tex`
